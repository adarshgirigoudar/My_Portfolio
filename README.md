# Adarsh CR7 Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my skills, services, projects, and contact information as a Java Developer.

## 🚀 Features

- Responsive navigation menu
- Animated hero section
- Services section
- Projects showcase
- About me section
- Contact information
- Social media integration
- Smooth scrolling
- Mobile-friendly design
- Interactive hamburger menu

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts (Montserrat)
